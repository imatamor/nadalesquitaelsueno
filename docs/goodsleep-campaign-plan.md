# Goodsleep Campaign Plan

## Stack confirmado
- Elementor
- Ultimate Addons for Elementor para header y footer
- Smart Slider 3 para hero
- Theme base `Maruri`
- Plugin custom `Goodsleep Elementor`

## Secciones
- `top`: hero slider nocturno con navegacion minima.
- `como-funciona`: bloque editorial construido en Elementor.
- `reglas`: bloque HTML con efecto de luz apagada/prendida.
- `historia`: widget `Historia Generator`.
- `historias`: widget `Historias List`.

## Plugin custom
- CPT publico para historias.
- Ajustes para Speechify y Mailjet.
- Catalogo de voces y tracks obtenido por API y filtrado por whitelist en admin.
- Ruta corta propia para compartir historias.
- Endpoints REST para generacion, favoritos, votos y listado.

## Integraciones
- Speechify para TTS.
- Mailjet para correo transaccional.
- Media Library para almacenar audios generados.

## Restricciones de experiencia
- La landing debe seguir siendo editable en Elementor.
- El theme no debe absorber logica especifica de campana.
- La web debe ser responsive y mantener una direccion visual cinematografica.
